How to Import on Xbox

  1. Download the .mcpack file from this site to your device
  2. Open Minecraft Bedrock Edition on your Xbox
  3. Go to "Create" → "My Packs"
  4. Select "Import" or "Browse"
  5. Locate the .mcpack file you downloaded
  6. Select it to import automatically
  7. Create a new world and enable your add-on in "Behavior Packs" & "Resource Packs"

💡 Alternative: Cloud Sync Method

If direct import doesn't work:

  1. Download the .mcpack to a cloud storage (OneDrive, Google Drive)
  2. On Xbox, open Edge browser and download the file
  3. The file saves to Downloads, then import via "My Packs"
